# Swagger schema split example

## DIRECTORY STRUCTURE

```
yaml/definitions/               definition files
yaml/paths/                     path files
yaml/parameters                 parameter files
```

## Installation

```
clone https://github.com/mole-chen/swagger-split.git
cd swagger-split
npm install -d --global gulp
npm install -d
```

## Usage

```
gulp
OR
gulp watch
```
